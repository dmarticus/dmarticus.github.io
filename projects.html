Here's some projects I've made

  • Scala Games. Games implemented for the console using functional programming principals in Scala
  • Slackbot. Provides a simple way to implement an interactive Slackbot in Go without having to expose an HTTP server.
  • Sentiment Analyzer CLI. An NLP CLI tool for analyzing the sentiment of a given phrase using a naive Bayes classifier with Laplace smoothing.
  • MonkeyLang Compiler. This package implements a compiler for an example programming language (similar to JS) named Monkey. It's my first foray into thinking about how compilers do things!
  • TypeScript Interpreter. A bootstrapped interpreter. For TypeScript, by TypeScript
  • Directed Acrylic Graph. This package is a mechanism to orchestrate goroutines and the order in which they run, using the semantics of a directed acyclic graph. Mostly, it was a chance to practice for interviews.
