Projects

Here are some of things I've worked on as a contributor...
  • Scala Exercises: An Open Source project for learning different technologies based in the Scala Programming Language
  • Higherkindness: Honest typed libraries and tools for anyone and everyone. Within the higherkindness project I've mostly worked on Mu; specifically these projects:
    • mu-scala. Mu RPC is a purely functional library for building RPC endpoint-based services with support for RPC and HTTP/2.
    • sbt-mu-srcgen. An sbt plugin that brings the ability to generate mu-scala protocols, services, and clients into your Scala program as part of the build.
    • mu-scala-examples. Demo implementations for Mu, a purely functional microservice library.
and as the primary author...
  • Scala games. Games implemented for the console using functional programming principles in Scala
  • dag4s. A library for rewriting and optimizing DAGs (directed-acyclic graphs) with FP Scala
  • Functional todo app. An example project that implements a REST API and DB connection using (hopefully) best practice functional Scala.
  • Slackbot. Provides a simple way to implement an interactive Slackbot in Go without having to expose an HTTP server.
  • Sentiment analysis CLI. A CLI tool for analyzing the sentiment of a given phrase using a naive Bayes classifier with Laplace smoothing.
  • Monkey. This project implements a compiler for an example programming language (similar to JS) named Monkey. It's my first foray into thinking about how compilers do things!
The rest of my stuff, none of which is particularly noteworthy, but serve as important milestones of my software development journey, can be found at my Github.
