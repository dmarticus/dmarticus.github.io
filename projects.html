Here's some projects I've made

  • Bloom filter. This package implements a concurrent and counting Bloom filter in Go.
  • MonkeyLang Interpreter. This package is an interpreter for an example programming language (similar to JS) named Monkey. It's my first foray into thinking about how compilers do things! My compiler project is coming soon
  • Directed Acrylic Graph. This package is a mechanism to orchestrate goroutines and the order in which they run, using the semantics of a directed acyclic graph. Mostly, it was a chance to practice for interviews.
  • Sentiment Analyzer CLI. An NLP CLI tool for analyzing the sentiment of a given phrase using a naive Bayes classifier with Laplace smoothing.
  • Whois Lookup. This is a tool for running whois lookups on websites. You can search via DNS or IP address.
