These are a few of my favorite things I made

  • Scala Games. Games implemented for the console using functional programming principals in Scala
  • dag4. A library for rewriting and optimizing DAGs (directed-acyclic graphs) with FP Scala
  • Functional Todo App. An example project that implements a REST API and DB connection using (hopefully) best practice functional Scala.
  • Slackbot. Provides a simple way to implement an interactive Slackbot in Go without having to expose an HTTP server.
  • Sentiment Analyzer CLI. An NLP CLI tool for analyzing the sentiment of a given phrase using a naive Bayes classifier with Laplace smoothing.
  • Monkey Compiler. This package implements a compiler for an example programming language (similar to JS) named Monkey. It's my first foray into thinking about how compilers do things!
